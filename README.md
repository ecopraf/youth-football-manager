# Youth Football Manager

La memoria digitale della squadra di calcio giovanile

## Panoramica

Youth Football Manager è un'applicazione web completa per la gestione di squadre di calcio giovanili. Permette di gestire giocatori, statistiche, presenze e molto altro.

## Struttura del Progetto

```
youth-football-manager/
├── backend/           # API backend (Node.js/Express)
├── frontend-v2/       # Frontend moderno (Vite/React)
└── README.md
```

## Tech Stack

- **Backend**: Node.js, Express
- **Frontend**: React (Vite)
- **Deploy**: Vercel

## Setup

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend-v2
npm install
npm run dev
```

## Funzionalità

- Gestione roster giocatori
- Statistiche individuali (presenze, minuti giocati, gol, assist)
- Categorie squadre (U14, etc.)
- Interfaccia mobile-friendly
- Dashboard con statistiche di squadra
